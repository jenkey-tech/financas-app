# App de Finanças Pessoais v5

## Como rodar

```bash
python financas_app.py
```

## Instalar gráficos melhores

```bash
pip install -r requirements.txt
```

## Novidades

- Lançamento repetido soma automaticamente para categorias acumulativas.
- Para categorias fixas, pergunta se deseja somar, substituir ou cancelar.
- Sugestão automática de valor baseada no histórico.
- Botão **Usar sugestão**.
- Ao clicar numa conta pendente/vencida, o formulário já tenta sugerir o valor.
- Projeção do mês: pago, não pago, pendente estimado e projeção final.
- Novo gráfico: cartões x gastos fixos.
- Visualização padrão abre no mês vigente, mesmo sem lançamentos.
